to run:

open in vscode
in terminal
npm install

in the root create a ".env" file. 
in the env file type:
(not in brackets)

TOKEN=[YOUR BOT TOKEN]
GUILD_ID=[YOUR SERVER ID]
CLIENT_ID=[YOUR BOT'S ID]


in terminall run:
node src/register-commands.js
nodemon

then ur done!
